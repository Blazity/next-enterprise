# Approved Component Library Index

**CRITICAL: All UI components MUST be sourced from this list. Creating custom components is NOT permitted.**

This document contains **492+ pre-approved components** from 8 vetted libraries. Before implementing ANY UI element, consult this index and use the appropriate component from the approved libraries.

---

## Library Priority Order

When selecting a component, check libraries in this order:

1. **shadcn/ui** - Foundation components (forms, dialogs, data display)
2. **Aceternity UI** - Effects, backgrounds, 3D, scroll animations
3. **Magic UI** - Text animations, buttons, device mockups
4. **Motion Primitives** - Morphing effects, text animations
5. **Animate UI** - Animated Radix ports, button primitives
6. **Eldora UI** - Device mockups, text effects
7. **Cult UI** - Neumorphic styles, texture effects
8. **ABUI** - Specialized components

## Decision Tree

When you need a component:

```
1. What category does it fall under?
   └─> Find category above

2. Is there an exact match?
   └─> YES → Use that component
   └─> NO → Find closest match

3. No close match exists?
   └─> Combine 2-3 approved components
   └─> NEVER create from scratch

4. Need customization?
   └─> Extend approved component's styles
   └─> Use design tokens from TOKENS.md
   └─> Do NOT modify component logic
```
